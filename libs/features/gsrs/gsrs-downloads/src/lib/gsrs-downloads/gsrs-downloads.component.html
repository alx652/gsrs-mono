G-SRS v3.1.2 (June 2025)
CHANGELOG

GSRS v3.1.2 has substantial code and organizational changes. Make sure to read the documentation on the new approaches to configuration before upgrading.

Highlighted new features include:
  • Spring Boot upgrade from 2.4.5 to 2.7.18
  • Revamp HOCON configuration; use environment variables and overrides
  • Configure GSRS extensions as map lists instead numerated lists to allow for overrides
  • Cross entity search
  • Structure search improvements
  • Nitrosamines flow chart
  • Product module Improvements
  • SSG1 Improvements
  • Impurities Improvements
Highlighted improvements and bug fixes include:
  • A new Exporter: SQL Exporter
  • ScheduledExportTask for non-substance services
  • Two InChIKeys for racemates and epimers
  • Show changes between two adjacent versions a substance
  • PDF download option
